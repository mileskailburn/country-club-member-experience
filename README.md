# Country Club Member Experience

Member engagement platform for Fort Collins Country Club (FCCC). Phase 0 discovery in progress.

**Stack (planned):** Next.js on Railway, Postgres/DuckDB, dbt-style SQL transforms, weekly CSV ingestion from ForeTees and Northstar.

See [PROJECT_BRIEF.md](./PROJECT_BRIEF.md) for full context, governance, phase scope, and current status.
